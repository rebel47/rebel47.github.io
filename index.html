AI Story Generator from Image

AI Story Generator from Image This Streamlit app uses AI to generate stories based on images. Users upload an image, and the app extracts detailed descriptions of the content using Google Gemini API. Then, it generates a short story based on the extracted description and converts the story into speech using Google Text-to-Speech (gTTS).
Key Features:
1. Image to Text: Converts the content of an uploaded image into a concise description using Google Gemini API.
2. Story Generation: Generates a short, realistic story based on the description of the image.
3. Text-to-Speech: Converts the generated story into an audio file and plays it back to the user.
4. Interactive: Users can upload an image, get a story, and listen to it in one seamless flow.
5. The app provides a fun and engaging way to create stories from images, making it ideal for creativity and entertainment purposes.

URL: scenescripter.streamlit.app
